# go-Examples

Go Examples contains some practice programs for learning the basics of Go Language. 
It contains programs on :

- [Closure]
- [Recursion]
- [Interface]
- [Struct]
- [GoRoutine]
- [Channels]
- [Channelsbuffering]
- [Packages]
- [ChannelSyncronization]
- [WorkerPool]
- [WaitGroup]
- [JSON]
- [Panic]
- [Defer]
- [Maps]
- [Range]
- [EnvironmentVariable]
- [HttpServer]
- [Currying]
- [FunctionComposition]
- [CollectionFunctions]
